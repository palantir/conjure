# Features

## HTTP endpoints

- GET, PUT, POST, DELETE
- Query parameters
- Path parameters
- Headers
- Cookie auth

## Types

You can define APIs using the following types:

- Complex types:
  - `list<T>`
  - `map<K, V>`
  - `optional<T>`
  - `set<T>`
  - Enum
  - Union

- Primitives
  - `any`
  - `bearertoken`
  - `binary`
  - `boolean`
  - `datetime`
  - `double`
  - `integer`
  - `rid`
  - `safelong`
  - `string`
  - `uuid`

- Migration types:
  - ExternalReference
